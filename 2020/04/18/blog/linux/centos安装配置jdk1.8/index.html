centos安装配置jdk1.8

清理之前的openjdk

yum -y remove java java-1.8.0-openjdk

下载jdk

  1. 手动去官网下载 或者使用自己的对象存储地址下载
  2. 下载tar或者rpm （jdk-8u181-linux-x64.tar.gz | jdk-8u172-linux-x64.rpm）

wget http://tar.matosiki.site/tart/jdk-8u181-linux-x64.tar.gz

如果是rpm使用命令安装

rpm -ivh jdk-8u172-linux-x64.rpm

上传到服务器 解压到固定目录

tar -zvxf jdk-8u181-linux-x64.tar.gz -C /usr/local/

更改解压后jdk主目录文件夹名称

mv /usr/local/jdk1.8.0_181 /usr/local/jdk1.8

配置环境变量

vi /etc/profile

source /etc/profile

测试java

java -version

快速安装方式

yum install java-1.8.0-openjdk* -y
