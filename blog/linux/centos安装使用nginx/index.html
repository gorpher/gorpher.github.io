centos安装使用nginx

预置环境

yum install -y gcc gcc-c++

yum -y install pcre-devel openssl openssl-devel

下载

wget http://tar.matosiki.site/tar/nginx-1.15.0.tar.gz

编辑配置文件

vi /usr/local/nginx/conf/nginx.conf

更改端口

  1. 静态文件转发
  2. 反向代理
  1. 负载均衡

启动

配置开机启动

更改下面配置文件

vi /etc/init.d/nginx

更改文件权限

chmod a+x /etc/init.d/nginx
